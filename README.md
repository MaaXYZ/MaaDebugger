# MaaDebugger

## Requirement

Python >= 3.11

## Installation

```bash
python -m pip install MaaDebugger
```

## Update

```bash
python -m pip install MaaDebugger MaaFW --upgrade
```

## Usage

```bash
python -m MaaDebugger
```

## Development of MaaDebugger itself

```bash
cd src
python -m MaaDebugger
```

or

Using VSCode, press F5 in the project directory.
